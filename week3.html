Les 3: Typografie en opmaak in HTML & CSS

1. Oefening: Typografie op schermen

  • Experimenteren met kleur, typografie en witruimte op een webpagina via BPUiEditor
  • Aanpassen van titels (h1–h6), paragrafen en lijsten
  • Testen van verschillende lettertypes, groottes, spatiëring en kleuren
  • Observeren van hoe typografie de leesbaarheid en visuele hiërarchie beïnvloedt
  • Maken van screenshots van het eindresultaat voor het leerplatform

2. Export en opmaak in HTML & CSS

  • Ontwerp gemaakt voor het scherm van de centrale stationshal
  • Export naar HTML en CSS met behulp van ChatGPT
  • Prompt gebruikt voor de export, bijvoorbeeld:
    • "Create 3 HTML files based on these screens that include CSS and the import of the fonts. The font is Roboto Slab."
  • Gegenereerde bestanden bewaard in een map en hoofdpagina genaamd index.html
  • Bestanden meegenomen naar de volgende les voor publicatie op GitHub en Netlify
