Les 6: Werken met Tailwind

Wat we hebben geleerd

Deze les ging over hoe we onze website konden omzetten naar Tailwind. We leerden hoe Tailwind werkt en hoe je bestaande CSS-styling kunt vervangen door Tailwind-klassen. Daarna hebben we onze logboek-site verder verbeterd in Visual Studio Code.

Wat we hebben gedaan

  • Tailwind toegevoegd aan ons project.
  • Oude CSS omgezet naar Tailwind-classes.
  • Layout en spacing verbeterd met utility classes.
  • Kleine aanpassingen aan de logboekpagina's.

Wat ik onthoud

  • Tailwind werkt zonder externe CSS-bestanden, alles gebeurt via classes.
  • Het is sneller om stijlen toe te voegen en aan te passen.
  • Je website wordt overzichtelijker omdat je minder losse CSS nodig hebt.
