Les 4: HTML & CSS - Typografie, Kleur en Ruimte

Inhoud van de les

  • Introductie tot HTML & CSS
  • Focus op typografie: lettertypes, grootte en hiërarchie
  • Kleurgebruik en contrasten voor betere leesbaarheid
  • Ruimte en marges: hoe tekst en elementen overzichtelijk plaatsen
  • Praktische oefening: HTML-pagina’s structureren en stijlen toepassen

Observaties tijdens de les

  • Het belang van consistente typografie voor overzicht
  • Kleur en contrast hebben grote invloed op leesbaarheid
  • Marges en witruimte zorgen voor visuele rust
  • Praktische oefeningen helpen de theorie te begrijpen

Besproken tools & bronnen

  • Google Fonts voor het kiezen en importeren van lettertypes
  • Visual Studio Code voor het schrijven van HTML & CSS
  • Online editor voor snelle tests (bv. BPUiEditor)
