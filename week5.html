Les 5: GitHub & Netlify

Wat we hebben gedaan

In deze les leerden we hoe we onze website online kunnen zetten. Eerst maakten we een repository aan op GitHub en plaatsten we onze bestanden erin. Daarna koppelden we deze repo aan Netlify zodat de website automatisch online komt te staan.
